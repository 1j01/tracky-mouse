Companion Software

Tracky Mouse pairs great with these other assistive technologies.

On-Screen Keyboards

  • Dasher (Windows/Linux, Free & Open Source) - I highly recommend trying this one out. It may look intimidating at first, but mainly you only need to memorize the order of the alphabet (in a spatial way) to become proficient with it.
  • OptiKey Pro (Windows, Free & Open Source) - Very solid. Everything you'd expect from an on-screen keyboard, plus tools for triggering different types of clicks and scrolls.
    âš ï¸ Note however that some of the features may conflict and cause Tracky Mouse to stop moving the mouse. Also OptiKey by default only allows dwell clicking on the keyboard, so it may need some configuration to allow clicking on it with facial gestures.

Voice Control

I don't have any specific recommendations for voice control software at the moment, but note that any scriptable voice control software should be able to start/stop Tracky Mouse using the command line interface.

Drawing

JS Paint can be configured with large buttons for use with Tracky Mouse.

JS Paint also includes Tracky Mouse, so you can enable Extras > Head Tracker to try out the Tracky Mouse UI, or Extras > Dwell Clicker to enable dwell clicking which is powered by Tracky Mouse but configured to work with JS Paint's UI specifically. If you're using the Dwell Clicker mode in JS Paint, make sure to turn off clicking in the Tracky Mouse desktop app. Also try View > Fullscreen and Extras > Speech Recognition.
